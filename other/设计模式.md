5. 装饰模式
```
装饰者模式就是用来动态的给对象加上额外的职责

Builder模式是一种创建型的设计模式. 旨在解决对象的差异化构建的问题.
装饰者模式是一种结构型的设计模式. 旨在处理对象和类的组合关系.
```

```
代理模式旨在为一个对象提供一个代理, 来控制对这个对象的访问.
```
